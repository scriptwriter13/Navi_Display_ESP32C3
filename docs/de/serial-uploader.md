# Serial Uploader Anleitung

Hier kannst du die Firmware direkt über deinen Browser auf den ESP32-C3 flashen.

## Voraussetzungen
- Ein Browser mit Web Serial Unterstützung (Chrome, Edge, Opera).
- Ein USB-C Kabel, das Daten übertragen kann.

## Schritte
1. Verbinde den ESP32-C3 mit dem PC.
2. Klicke auf "Connect".
3. Wähle den Port aus.
4. Klicke auf "Install".
